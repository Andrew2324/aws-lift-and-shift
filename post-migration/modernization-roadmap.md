# Modernization Roadmap

## Phase 1 – Lift and Shift (Completed)
- VM migrated to AWS
- Infrastructure defined via Terraform
- Minimal changes to application

## Phase 2 – Resilience
- Add Application Load Balancer
- Introduce Auto Scaling Group
- Move to private subnets

## Phase 3 – CI/CD
- Automate application deployments
- Blue/Green or rolling deployments
- Infrastructure pipelines

## Phase 4 – Containerization
- Containerize application
- Deploy to ECS or EKS
- Improve scalability and portability
